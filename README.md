Bu dasturda sodda arifmetik amallar va Math kutubxonasidagi bir nechta amallarni bajarish uchun mo'ljallangan.

- Sodda arifmetik amallar
  - qo'shish
  - ayirish
  - ko'paytirish
  - bo'lish
  - qoldiq
 
- Math kutubxonasidan
  - `Math.Round()`
  - `Math.BigMul()`
  - `Math.Max()`
  - `Math.Min()`
